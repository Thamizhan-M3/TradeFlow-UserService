# TradeFlow User Service

Owns authentication-ready user records and wallet balance data.

## Run

```bash
npm install
npm run dev
```

## Endpoints

- `GET /health`
- `POST /api/v1/users/register`
- `POST /api/v1/users/login`
- `GET /api/v1/users/:id`
- `PATCH /api/v1/users/:id/wallet`
